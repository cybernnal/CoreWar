# CoreWar

## 42 corewar project

Build: `make`

Usages: 

  `./asm asm_champ.s ; ./corewar asm_champ.cor [-v visual need full screen] [-d N	to dump memory after N cycle]`


The asm binary compile your asm file (.s) to corewar file (.cor).
